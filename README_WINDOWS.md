# 🖥️ Aegis for Windows PC

## ⚡ クイックスタート（3ステップ）

### 1️⃣ ダウンロード
[**こちらをクリック**](https://github.com/aka209859-max/-Aegis/archive/refs/heads/main.zip) して最新版をダウンロード

### 2️⃣ 解凍
ダウンロードした `Aegis-main.zip` を右クリック → 「すべて展開」

### 3️⃣ 起動
`index.html` をダブルクリック！

---

## ✨ 特徴

- ✅ **インストール不要**: ダウンロードして即起動
- ✅ **サーバー不要**: ローカルファイルを開くだけ
- ✅ **完全プライバシー**: データは一切外部送信されません
- ✅ **オフライン対応**: 初回接続後はネット不要（CDNキャッシュ）
- ✅ **軽量**: 単一HTMLファイル（約40KB）

---

## 📋 使い方

### 診断の流れ
1. トップページで診断を選択（未払いリスク or デジタル遺産）
2. 質問に回答（5問 or 4問・約2分）
3. スコアとXAI（計算式）を確認
4. 処方箋（改善命令）を実行

### 結果の保存
- **スクリーンショット**: Windows + Shift + S
- **ブラウザ印刷**: Ctrl + P → PDFとして保存
