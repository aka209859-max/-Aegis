# Aegis - Windows PC用インストールガイド

## 📥 ダウンロード方法

### オプション1: GitHubから直接ダウンロード（推奨）
1. https://github.com/aka209859-max/-Aegis にアクセス
2. 緑色の「Code」ボタン → 「Download ZIP」をクリック
3. ダウンロードした `Aegis-main.zip` を解凍
4. 解凍したフォルダ内の `index.html` をダブルクリック

### オプション2: 単一ファイルダウンロード
1. https://github.com/aka209859-max/-Aegis/blob/main/index.html にアクセス
2. 右上の「Raw」ボタンをクリック
3. ブラウザで「名前を付けて保存」（Ctrl + S）
4. 保存した `index.html` をダブルクリック

---

## 🖥️ 動作要件

### 必須
- **OS**: Windows 10/11
- **ブラウザ**: Microsoft Edge（推奨）/ Google Chrome / Firefox
- **インターネット**: 初回起動時のみ必要（CDNライブラリ読み込み）

### 推奨
- **画面解像度**: 1366×768 以上
- **RAM**: 4GB 以上

---

## 🚀 起動方法

### 方法1: ダブルクリック起動（最も簡単）
1. `index.html` を右クリック
2. 「プログラムから開く」→「Microsoft Edge」を選択
3. 診断開始！

### 方法2: デスクトップショートカット作成
1. `index.html` を右クリック → 「ショートカットの作成」
2. ショートカットをデスクトップに移動
3. アイコンをダブルクリックで即起動

### 方法3: 既定のブラウザで開く
1. `index.html` をダブルクリック
2. 既定のブラウザ（Edge/Chrome等）で自動起動

## 🔧 トラブルシューティング

### 問題1: ページが正しく表示されない
**原因**: インターネット接続がない、またはCDNブロック

**解決策**:
1. インターネット接続を確認
2. セキュリティソフト・ファイアウォールの設定を確認
3. 別のブラウザで試す（Edge → Chrome など）

### 問題2: 「このサイトは安全ではありません」警告
**原因**: ローカルファイルを開いているため（正常動作）

**解決策**:
- 警告を無視して「続行」をクリック
- これは正常な動作です（サーバー不要のため）

### 問題3: スタイルが崩れている
**原因**: Tailwind CSSが読み込まれていない

**解決策**:
1. ブラウザのキャッシュをクリア（Ctrl + Shift + Delete）
2. ページをリロード（F5）
3. インターネット接続を確認
